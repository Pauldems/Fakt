================================================================================
                      FAKT - GUIDE COMPLET DES FONCTIONNALITÉS
================================================================================

FAKT est une application mobile de facturation spécialement conçue pour les 
hébergeurs touristiques (Airbnb, chambres d'hôtes, locations saisonnières).

================================================================================
🔑 SYSTÈME D'ACTIVATION
================================================================================

ACTIVATION INITIALE :
- Code d'activation à 16 caractères (format: XXXX-XXXX-XXXX-XXXX)
- Validation en temps réel via Firebase
- Un code = un seul appareil (sécurité renforcée)
- Sauvegarde locale + cloud pour fonctionnement hors-ligne

TYPES D'ABONNEMENTS :
✓ Essai gratuit     : 7 jours
✓ Mensuel          : 1 mois (4,99€)  
✓ Trimestriel      : 3 mois (12,99€)
✓ Annuel           : 1 an (19,99€)
✓ Vie              : Permanent (49,99€)

FONCTIONNALITÉS :
- Extension d'abonnement possible
- Surveillance en temps réel du statut
- Blocage automatique si code désactivé
- Mode dégradé si pas de connexion
